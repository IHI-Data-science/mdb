8.1. Aperçu¶

L’application Mosquito DB est utilisée pour la collecte de données sur le terrain et permet également d’afficher les données d’expériences particulières lors du travail sur le terrain.

Requirements: | Système d’exploitation OS « Android – Version <4.0” | Téléphones intelligents ou tablettes | Accès à Internet – Téléchargement et téléchargement de projets | Installer Android Package Kit (APK) - pour le fichier Mosquito DB
